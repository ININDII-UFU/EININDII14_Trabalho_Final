# Pacote de apoio — Laboratório de controle de nível

Conteúdo:

- `ROTEIROS_LAB_CONTROLE_NIVEL.md`: documento mestre;
- `mapa_modbus_unificado.csv`: endereços padronizados;
- `variaveis_padrao.csv`: nomes, tipos e valores iniciais;
- `parametros_conexao.json`: IPs, portas, Unit ID, escalas e PID inicial;
- `modelo_relatorio_grupo.md`: modelo de entrega;
- `assets/`: símbolos para marcação das capturas;
- `grupo_1` a `grupo_4`: checklists resumidos.

Os arquivos nativos de OpenPLC, Open ModSim e FUXA devem ser exportados e testados na versão instalada no laboratório.
